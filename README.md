# User input in browser and node

## Method browser

This is a simple HTML file that includes a JavaScript file called main.js. Within main.js we use JavaScript's `prompt` function to use the browser's default popup window to request input from the user. 

### Running instructions

Clone this repository and open index.html in your browser.

## Method Node

This uses an npm package called `prompt`

### Running instructions

Clone this repository to your computer and run `$ npm install` in the method-node directory. Once this has completed, run the script by running `$ node input.js`.
